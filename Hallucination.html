HALLUCINATION

Hallucination is what I consider to be my true first album (no disrespect to Depths of the Cave or Nerrolad). It has a couple official releases:

  • 5-Track Release (original Spotify release)
  • 8-Track Release (original Bandcamp release. Lost media)
  • 10-Track Release (original BandLab release)
  • 11-Track Release (modern release. All tracks + enchanced audio quality)

This page will primarily be talking about the 11-Track Release. I'm still not entirely sure how there ended up being so many different tracklists. I assume I just kept on forgetting to add track after track until I finally got my act together with the 11-Track release.
